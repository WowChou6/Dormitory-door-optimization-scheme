*项目名称：ESP8266 RFID 开门工具

*项目描述： 本项目使用 ESP8266-NodeMCU 开发板和 RC522 RFID 模块实现了一个简单的 RFID 卡片读取器。你可以使用这个设备来读取RFID卡片的UID，并将其显示在串口监视器上，同时你可以将读取到的UID与已知卡片UID进行比对以进行身份验证，进而实现刷卡开门的效果。

*所需材料：
1. ESP8266-NodeMCU 开发板
2. RC522-RFID 模块
3. 0.96寸OLED 模块
4. 2节18650电池
5. 5v降压模块
6. 舵机（如SG90等，可根据实际情况选择不同型号的舵机）
7. 外壳（可以使用纸板或3D打印等）
8. 其他（15根以上的母对母杜邦线、一定数量的公对公和公对母杜邦线、绝缘胶带、USB数据线、电脑、鼠标、键盘、桌子、椅子、门、手、脑子等）

*连接硬件：请确保正确连接 ESP8266-NodeMCU 开发板与各个模块，如文件夹内的 [接线图] 所示。

*软件设置：
1. 下载并打开Arduino IDE。
2. 确保你已安装ESP8266开发板支持。
3. 安装相关库文件，方法是在Arduino IDE中选择“工具”>“管理库”并搜索相应的库文件，然后点击安装。
4. 安装串口驱动（请根据你的开发板的串口驱动芯片来选择。如：CH340、CP2102等）
4. 在”工具“菜单中选择开发板和其所在的端口（开发板选择：”NodeMCU 1.0（ESP-12E Module)";端口可在设备管理器中查看）

*上传代码：
1. 打开 Arduino 项目文件（.ino文件）。
2. 点击“上传”按钮，将项目代码上传到ESP8266-NodeMCU开发板。

*读卡操作步骤：
1. 将 ESP8266-NodeMCU 开发板连接到电脑并上传代码。
2. 打开串口监视器。（之后可能需要按下开发板的复位按键）
3. 你应该看到串口监视器上的初始化消息，这表示RFID读卡器已准备好工作。
4. 将一个RFID卡片或标签放在RC522模块的附近。
5. RFID模块将读取卡片的UID，并将其显示在串口监视器上。
6. 卡片的UID将以十六进制格式显示，例如："Card UID: 4A 6D 2B 8E"。

*最后一步：将其稳妥地安装到你的door上。

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
*注意事项：
0. 本装置专为 【相机型】 门锁打造，请根据自己的门锁类型进行相应改动。
1.开始制作前请务必【考虑周全】（包括但不限于：线材长度、装置大小、安装位置及可靠性、方案科学性和可行性、开门所需的力度等）
2. 确保RFID模块与卡片之间没有或尽可能减少物理障碍，以确保正常读取。
3. RFID模块与开发板的接线应尽可能的【短】（10cm左右），否则其工作状态将变得不稳定（别问我为什么知道😅）。
4. 当卡片不再在RFID模块附近时，UID信息将不再显示，但模块仍然处于工作状态。
5. 请确保供电电压为【稳定的5V】。
6. 使用电池进行供电时【请务必注意】其潜在的风险，并做好充分的安全措施。

*项目扩展：
此项目只是一个基本示例，你可以根据自己的需求扩展功能。例如将卡片信息录入手机以利用NFC功能开门、利用该开发板的 WiFi 功能实现远程的开门或状态监控操作、利用OLED屏幕进行个性化内容显示、增添门铃功能、优化供电方案等等等等等等等。
